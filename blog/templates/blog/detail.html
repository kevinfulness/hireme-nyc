challenge

Google Maps list-making requires manual saving, sorting, and note taking. Users who are researching places for an outing or trip need to manually search for places, create a list, add notes and a title before sharing with friends. The challenge was to simplify this process into a single prompt using Gemini.

considerations

While the possibilities of AI prompting are endless, we were under a time crunch -- leadership wanted a working PoC in dogfood in 3 months.

solution

After insights from some rapid user testing, we refined early concepts into an input module on the You Tab (home of saves & lists). We let suggestion chips do some user education heavy lifting, reducing visual noise while providing real-world use-cases for the tool. When prompted, the tool would return an unsaved list, ready for user customization, with AI generated emoji, title, hero image, description, and notes.

impact

Me and a small xfn team took this tool from 0 to Googe-wide dogfood in 3 months. We used dogfood as an additional source of user insights, while pushing towards a general release. Meanwhile, the product provides a jumping off point for additional Gen AI features.
